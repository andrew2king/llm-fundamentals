## 二、课程大纲（4 大模块）

### 模块一：Agent 基础概念（4 课时）

**课时 1：什么是 Agent？**
- Agent 定义与大模型的区别
- Agent 的核心能力：自主决策、工具使用、记忆管理
- Agent 应用场景全景图

**课时 2：Agent 核心架构**
- 感知层：输入理解与环境感知
- 决策层：规划与推理
- 行动层：工具调用与执行
- 反馈层：结果评估与迭代

**课时 3：核心范式解析**
- ReAct（推理+行动）
- Plan-and-Execute（规划执行）
- Reflection（反思改进）
- 范式选择指南

**课时 4：技术路线选择**
- Agent vs Chain vs RAG
- 何时使用 Agent？
- 开源框架对比：LangChain vs LlamaIndex

---

### 模块二：LangChain 核心组件（6 课时）

**课时 5：LLM 与 Prompt 模板**
- LLM 抽象层设计
- PromptTemplate 进阶用法
- Few-shot 提示技巧
- 多模型切换策略

**课时 6：Memory 记忆系统**
- 对话记忆：ConversationBuffer
- 滑动窗口：ConversationBufferWindow
- 摘要记忆：ConversationSummary
- 向量记忆：VectorStoreMemory

**课时 7：Tool 工具定义与调用**
- Tool 接口设计
- 内置工具使用
- 自定义工具开发
- 工具调用流程解析

**课时 8：Chain 链式调用**
- LLMChain 基础
- SequentialChain 顺序链
- RouterChain 路由链
- TransformChain 数据转换

**课时 9：OutputParser 结构化输出**
- PydanticOutputParser
- Structured Output with Tools
- 自定义 Parser
- 错误处理与重试

**课时 10：回调与调试**
- Callbacks 机制
- LangSmith 集成
- Token 消耗监控
- 调试最佳实践

---

### 模块三：Agent 开发进阶（6 课时）

**课时 11：AgentExecutor 原理解析**
- AgentExecutor 架构
- 执行流程详解
- 超时与停止条件
- 自定义 Agent 类

**课时 12：自定义 Tool 开发实战**
- Tool 开发规范
- 输入验证与类型提示
- 错误处理策略
- Tool 测试方法

**课时 13：Multi-Agent 协作基础**
- 多 Agent 架构设计
- 角色分工与协作
- LangGraph 入门
- 简单协作案例

**课时 14：错误处理与重试策略**
- 常见错误类型
- 重试机制设计
- 降级策略
- 用户友好提示

**课时 15：流式输出与用户体验**
- Streaming 原理
- 前端集成方案
- 进度展示设计
- 取消与中断处理

**课时 16：成本优化技巧**
- Token 计费原理
- Prompt 压缩技巧
- 缓存策略
- 模型选择优化
