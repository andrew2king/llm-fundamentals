## 二、什么是 Agent？

### 2.1 定义

**Agent（智能体）** 是一个能够**自主感知环境、做出决策、执行行动**的智能系统。

用一句话概括：

> Agent = LLM + 规划能力 + 工具使用 + 记忆

### 2.2 与普通 LLM 应用的区别

| 特性 | 普通 LLM 应用 | Agent |
|------|--------------|-------|
| 交互模式 | 一问一答 | 多轮自主执行 |
| 工具使用 | 无或简单调用 | 主动选择和使用工具 |
| 记忆能力 | 无状态 | 持久记忆与上下文 |
| 决策能力 | 无 | 自主规划与决策 |
| 目标导向 | 被动响应 | 主动完成任务 |

### 2.3 一个形象的比喻

**普通 LLM 应用** 就像一个**咨询顾问**：
- 你问什么，它答什么
- 不会主动帮你做事
- 每次对话都是新的开始

**Agent** 就像一个**智能助理**：
- 你给一个目标，它自主完成
- 会主动使用工具（搜索、计算、调用 API）
- 记得住之前的对话
- 遇到问题会自己想办法解决

## 三、Agent 的核心能力

### 3.1 四大核心能力

```
┌─────────────────────────────────────────┐
│                 Agent                    │
│                                         │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐ │
│  │ 感知能力 │→│ 决策能力 │→│ 行动能力 │ │
│  └─────────┘  └─────────┘  └─────────┘ │
│       ↑                          │      │
│       └──────── 记忆能力 ────────┘      │
└─────────────────────────────────────────┘
```

**1. 感知能力**
- 理解用户意图
- 感知环境状态
- 接收外部信息

**2. 决策能力**
- 任务规划
- 步骤分解
- 策略选择

**3. 行动能力**
- 工具调用
- API 执行
- 结果输出

**4. 记忆能力**
- 短期记忆（对话上下文）
- 长期记忆（知识存储）
- 工作记忆（任务状态）

### 3.2 能力示例

**用户任务**：「帮我分析一下 OpenAI 最近一年的股价走势」

| 阶段 | Agent 行为 | 核心能力 |
|------|-----------|---------|
| 感知 | 理解需要获取股票数据并分析 | 感知能力 |
| 决策 | 规划：获取数据 → 绘制图表 → 分析趋势 | 决策能力 |
| 行动 | 调用股票 API、绘图工具 | 行动能力 |
| 记忆 | 记住用户关注的是 OpenAI，后续可引用 | 记忆能力 |

## 五、Agent 的本质：从 Chat to Action

### 5.1 演进路径

```
传统软件 → LLM 应用 → Agent → Multi-Agent
   │           │          │          │
 手动操作    对话交互   自主执行    协作完成
```

### 5.2 核心转变

| 维度 | 传统 LLM | Agent |
|------|---------|-------|
| 交互方式 | Chat（聊天） | Action（行动） |
| 用户角色 | 驱动者 | 指挥者 |
| LLM 角色 | 回答者 | 执行者 |
| 目标 | 回答问题 | 完成任务 |

### 5.3 一个实际例子

**用户需求**：「帮我找出公司本月销售额最高的前 10 个客户，并生成分析报告」

**传统 LLM 应用**：
```
用户：帮我找出公司本月销售额最高的前 10 个客户
LLM：我无法直接访问你的数据库，请提供数据...
用户：（手动导出数据，粘贴给 LLM）
LLM：根据你提供的数据，销售额最高的是...
```

**Agent**：
```
用户：帮我找出公司本月销售额最高的前 10 个客户，并生成分析报告
Agent：好的，我来处理这个任务。
      [调用数据库工具] 查询本月销售数据...
      [分析数据] 排序并筛选 TOP 10...
      [生成报告] 创建分析报告...
      已完成！TOP 10 客户如下...完整报告已发送至你的邮箱。
```

## 六、本课小结

### 6.1 关键要点

1. **Agent 定义**：LLM + 规划 + 工具 + 记忆
2. **核心区别**：从被动响应到主动执行
3. **四大能力**：感知、决策、行动、记忆
4. **应用场景**：企业、个人、开发者全覆盖
