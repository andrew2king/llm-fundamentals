## 学习路线

```
第一阶段：基础入门（2-4周）
├── Python 基础
├── LLM 基本概念
├── Prompt 工程
└── API 调用实践

第二阶段：框架实战（4-6周）
├── LangChain 核心
├── RAG 系统构建
├── Agent 开发
└── 向量数据库

第三阶段：进阶应用（6-8周）
├── Multi-Agent 系统
├── 模型微调
├── 部署优化
└── 生产实践

第四阶段：领域深耕（持续）
├── 行业应用
├── 前沿论文
├── 开源贡献
└── 商业落地
```

## 避坑指南

❌ **不要一上来就学 Transformer 原理**
✅ 先会用，再理解原理

❌ **不要只看不练**
✅ 每学一个概念就写代码

❌ **不要追最新论文**
✅ 先掌握核心技能

❌ **不要孤军奋战**
✅ 加入学习社群，一起进步
